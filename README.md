# 🧠 Rotinacognia

Plataforma web para gestão de rotinas e desenvolvimento cognitivo.

🔗 **Acesse o projeto:**  
[https://rotinacognia.lovable.app](https://rotinacognia.lovable.app)

---

## 📋 Sobre o projeto

O **Rotinacognia** foi desenvolvido para auxiliar no gerenciamento de tarefas diárias com foco no aprimoramento cognitivo.  
A ferramenta permite organizar rotinas de forma intuitiva, ajudando usuários a manterem consistência, produtividade e clareza mental.

---

## 🎯 Funcionalidades principais

- ✅ Cadastro e organização de tarefas diárias  
- 📅 Planejamento de rotinas semanais  
- 🧩 Interface simples e responsiva  
- 🔔 Lembretes visuais para acompanhamento  
- 📊 Acompanhamento de progresso (em desenvolvimento)

---

## 🛠️ Tecnologias utilizadas

- **Frontend:** React / Vite  
- **Estilização:** TailwindCSS  
- **Deploy:** Lovable.app  
- **Versionamento:** Git + GitHub
